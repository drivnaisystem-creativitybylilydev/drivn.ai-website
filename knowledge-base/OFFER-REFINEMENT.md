## The Solution: Tier 1 Focus

### Tier 1 (5/6 - 6/6) — Lead Capture + Reputation System

**Setup fee:** $800  
**Monthly retainer:** $100  
**Timeline:** 10–14 days to live

**What's included:**
- Professional website (mobile-optimized, Google-friendly)
- Lead capture form (auto-saves to CRM dashboard)
- Basic CRM dashboard (see inquiries, track status, notes)
- Auto Google review requests (email template, 30-day auto-send)
- Monthly performance report (leads captured, conversion rate)

**What it solves:**
- Missed leads going to competitors → Captured in CRM
- Manual review requests → Automated
- No visibility into lead quality → Dashboard shows everything

**Why this first:**
- Fastest to build (1–2 weeks)
- Validates core problem (missed leads, no system)
- Creates repeatable playbook
- Can demo live within days

**Success metric:** Prospect gets 5–10 leads/month in CRM, can see where they're coming from
